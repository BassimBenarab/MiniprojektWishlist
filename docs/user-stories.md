# User Stories — MiniprojektWishlist

Som bruger vil jeg kunne oprette en konto (eller indtaste mit navn),
så jeg kan have min egen ønskeseddel.

Som bruger vil jeg kunne oprette en ny ønskeseddel,
så jeg kan samle mine ønsker ét sted.

Som bruger vil jeg kunne tilføje et ønske med titel, beskrivelse og evt. link,
så jeg kan huske, hvad jeg ønsker mig.

Som bruger vil jeg kunne se en liste over mine ønsker,
så jeg nemt kan få overblik over dem.

Som bruger vil jeg kunne redigere et ønske,
så jeg kan ændre detaljer, hvis jeg fx finder et nyt link eller en bedre beskrivelse.

Som bruger vil jeg kunne slette et ønske,
så jeg kan fjerne noget, jeg ikke længere ønsker mig.

Som bruger vil jeg kunne se hele min ønskeseddel i browseren,
så jeg kan vise den til andre eller selv tjekke den.
